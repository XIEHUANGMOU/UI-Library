## ✨ 核心特性

- 🎨 **视觉美化**：内置彩虹边框、自定义背景图与图标支持。
- 🖥️ **高 DPI 适配**：自动缩放 UI 尺寸，完美兼容各种分辨率设备。
- 🧩 **组件化架构**：支持在 `Tab` 或 `Section` 中无限嵌套组件。
- ⌨️ **全局快捷键**：几乎所有组件均支持绑定独立的快捷键。
- 🔔 **全局通知系统**：自带支持图标与音效的 Toast 通知推流。
